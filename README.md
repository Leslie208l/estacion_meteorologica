Descripción del Proyecto

Este proyecto consiste en el desarrollo de una **Estación Meteorológica Automatizada** capaz de monitorear y registrar datos climáticos en tiempo real. El sistema utiliza un microcontrolador ESP32 conectado a diferentes sensores para medir variables ambientales como temperatura, humedad, velocidad del viento y radiación solar.

Los datos recolectados son enviados mediante una API a un servidor donde se almacenan en una base de datos SQL. Posteriormente, la información se visualiza en una aplicación móvil desarrollada con React Native, permitiendo consultar lecturas en tiempo real, visualizar gráficos de los datos y gestionar los sensores desde una interfaz intuitiva.

El objetivo del sistema es facilitar el monitoreo de condiciones climáticas para análisis ambiental y apoyo en la toma de decisiones.
